# Folder Plan: ref/src/dashboard

## Architecture
- Scope: ref/src/dashboard
- Function: Device-page subcomponents extracted from DeviceDashboard.jsx during the layout reorg — DeviceStatusBar (区 1: board id + USB/WiFi labels + independent USB and WiFi status chips + 手动 USB 串口重新扫描), ChannelMatrixCard (区 2: 每渠道一行，展示 / 绑定 / 切换形象), BoardButtonPanel (区 3: larger non-overlapping SVG callout map + compact button rows + full button-map USB OTA trigger), VoiceAssistantPanel (区 4: 折叠态语音开关 + 共享表单控件节奏的会话续接 + mock/设备语音注入状态 + 启动收听), DashboardActionsMenu (PageShell 右上 actions 菜单).
- Sync: if this folder changes, update this file immediately.
- Reuse: consumed only by `ref/src/DeviceDashboard.jsx`. State comes from `shell/DeviceContext.jsx` via `useDeviceContext()`; user notices come from `shell/ToastStack.jsx` via `useToast()`. Channel-switch confirm modal continues to live at `ref/src/ChannelSwitchConfirmModal.jsx`.
